# seed 数据来源记录

这份 seed 数据是我先手工整理出来的一小批样本，目的不是追求规模，而是先把“真实技术文档数据怎么落到这个项目里”这件事做顺。

当前这一批样本只用了官方来源：

- Git 官方文档
  - https://git-scm.com/docs/git-rebase
  - https://git-scm.com/docs/git
  - https://git-scm.com/docs/git-reset
- GNU grep 官方手册
  - https://www.gnu.org/software/grep/manual/html_node/File-and-Directory-Selection.html
  - https://www.gnu.org/software/grep/manual/html_node/Output-Line-Prefix-Control.html
  - https://www.gnu.org/software/grep/manual/html_node/Matching-Control.html

我这里没有直接照抄原文，而是做了简化和转述，先整理成适合问答和检索的小 chunk。后面如果继续扩数据，我希望保持几个原则：

- 只用稳定、可靠的官方来源
- 一条问答尽量只对应少量核心文档块
- 问题类型不要只停留在概念题
- 把来源 URL 一起保留下来，方便后面回查
