# 数据集笔记

先只做一个比较窄的范围，不想一开始铺太大。

目前先放在这些方向里选：

- Git
- Linux 命令行
- Docker
- PyTorch

## 文档字段

先保留这些：

- `doc_id`
- `title`
- `text`
- `source`
- `domain`

后面如果有需要再补：

- `section_path`
- `tool_name`
- `version`
- `task_type`

## 问题类型

至少要有这几类：

- 概念题
- 步骤题
- 参数题
- 排错题

## 看结果时先看什么

- 有没有检索错
- 有没有检索到了但没答出来
- 有没有漏步骤
- 有没有把相似命令搞混
